This is a minimal Next.js (App Router) + Tailwind + TypeScript package that renders the Sales Scout preview UI.

## Quickstart
```bash
npm install
npm run dev
# open http://localhost:3000
```

## Deploy
- Push to GitHub (e.g. melexis-sales-scout-preview-app)
- Import the repo in Vercel (framework auto-detected as Next.js)
- Build command: `next build` (default)
- Output: `.next` (default)
